Sample Project with Math

2025 • Tools: HTML, CSS, MathJax

This page shows how LaTeX renders on GitHub Pages using MathJax. Inline math like $E=mc^2$ or $\\alpha + \\beta = \\gamma$ works, and display math too:

$$ \\nabla \\cdot \\vec{E} = \\frac{\\rho}{\\varepsilon_0} $$

Derivation sketch

Consider the loss function $$ \\mathcal{L}(\\theta) = \\frac{1}{N} \\sum_{i=1}^N \\bigl(f_\\theta(x_i) - y_i\\bigr)^2, $$ its gradient is $$ \\nabla_\\theta \\mathcal{L}(\\theta) = \\frac{2}{N} \\sum_{i=1}^N \\bigl(f_\\theta(x_i) - y_i\\bigr)\\,\\nabla_\\theta f_\\theta(x_i). $$
